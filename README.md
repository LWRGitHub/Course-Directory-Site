#  Course Directory Website
Fun Website build with React Router that talks about teaching different technologies. With lists of courses & teachers. This site is mainly to shows off my skill with React Router on desktop & mobile.

**View Website:** [Here](https://lwrgithub.github.io/Course-Directory-Site/#/)

## Technologies Used
- JavaScript
- CSS
- React
- Rect Router
- React HashRouter
- Google Web Fonts
- [GitHub Pages](https://lwrgithub.github.io/Course-Directory-Site/#/)

## Install/Setup

**Install**
1. install dependencies
```
yarn install
```
2. If you want to use GitHub pages 
***NOTE: make sure to change the files to reflect your GitHub info:*** [Learn How](https://blog.usejournal.com/how-to-deploy-your-react-app-into-github-pages-b2c96292b18e)
```
npm install gh-pages --save-dev
```


**Run:** localhost:3000
```
yarn start
```

**Push Info**
1. if using GitHub Pages do build first
```
npm run deploy
```
2. Push
